LEG V
THE API STRIKES BACK

It is a period of free access to all endpoints.

Authorization has not yet been implemented.

Authentication may never be.

If the Rebel Alliance succeeds, all API endpoints (other than those required for authentication) will require login for access.

To ensure the privacy and safety of our users, no incident reports will be publicly accessible.

Incidents and Users will be returned as follows:

<< Incident >>
{
    "id"           int
    "severity"     int {0, 1, 2, 3}
    "status"       int {0, 1, 2}
    "description"  string
    "location"     string
    "created_at"   string (ISO 8601)
    "user"         << User >>
}

<< User >>
{
    "id"             int

The API endpoints are as follows:

GET  /incidents
